# AGENTS

## Rules
- Plan first: write a short implementation plan before making non-trivial changes.
- Contracts first for new features: define or update types/interfaces and data contracts before UI wiring.
- Do not invent endpoints: only use endpoints that already exist in the repo or are explicitly specified.

## Commands To Run
- `npm run dev`
- `npm run lint`
- `npm run build`
- `npm run start`
